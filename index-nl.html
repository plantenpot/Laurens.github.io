Laat ik mij mijzelf voorstellen

Ik ben Laurens van Baardewijk, ik ben 18 jaar oud en woon in Nederland. Momenteel studeer ik Software Development op SintLucas te Eindhoven. De eerste keer ooit dat ik een script had gemaakt, was voor een game op Roblox. Roblox is een website vol met games gemaakt door gebruikers van de website. Ik vroeg mij altijd al af hoe mensen zulke games maakten en hoe ik mijn eigen kon maken. Als eerste probeerde ik wat willekeurige Engels woorden te typen in de hoop dat het iets zal doen. Natuurlijk hebben die scripts nooit gewerkt. Ik ging op YouTube opzoek naar tutorials over hoe ik bepaalde dingen in scripts kon maken. Bijvoorbeeld hoe ik een blokje van kleur kon veranderen. Na enkele YouTube tutorials te hebben gevolgd kon ik mijn eigen mini-game maken. Toen ik zag dat alles werkte zoals ik het in gedachte had, kreeg ik erg de motivatie om meer te leren en maken. Ik ging weer naar YouTube om meer te leren en maakte meer gecompliceerde games over de maanden. Helaas na een update van Roblox om exploiters tegen te gaan, werken de scripts niet meer en de games dus ook niet meer.

Na meerdere Roblox games te hebben gemaakt, kwam ik met het idee om een eigen website te maken. Ik wist nog niet hoe dit moest dus weer ging ik naar YouTube om tutorials te volgen. Ik kwam erachter dat HTML en CSS helemaal niet zo moeilijk waren om te leren. Op de eerste of tweede dag had ik al mijn eigen simpele website gebouwd. Als ik het mij goed herinner was de website bedoeld als kleine portfolio over mijzelf, maar gebruikte ik het meer om te leren hoe ik een website moest maken. Nadat ik klaar was met de basics van de website wilde ik verder gaan met een login scherm. Maar voor een login scherm heb je ook een database nodig en na meerdere YouTube tutorials te bekijken lukte het mij nog niet om dit werkende te krijgen. Het was simpelweg te complex voor mij toen de tijd.

Nadat ik meerdere keren had geprobeerd mijn login scherm werkende te krijgen, besloot ik te stoppen voor nu en door te gaan naar Unity games. Ik kwam erachter dat dat gemaakt moest worden in C# en dat dat nog helemaal niet zo makkelijk was. De allereerste game die ik in Unity gemaakt had, was een simpel bal race spel. Ik heb die game kunnen maken met behulp van de YouTube tutorials van Brackeys. Terwijl ik de game maakte speelde ik met de Unity physics, particle effect, UI's, etc. Ik was best blij toen ik de game af kreeg na een paar dagen. Het was een vrij simpele game. Het doel was om door verschillende levels te gaan en nieuwe highscores te maken. Nadat je alle levels had gespeeld kon je ze opnieuw spelen om je vorige highscore te verslaan. Ik heb veel geleerd van die game, maar doen ik meer gecompliceerde games probeerde te maken kwam ik erachter dat er nog genoeg te leren was. Op dat moment besloot ik na de middelbare school Software Development te gaan studeren.

In het vierde jaar van de middelbare school ben ik een hackers cursus online gaan volgen voor de lol. Ze hebben mij daarin geleerd hoe ik websites kon exploite en SQL databases kan binnendringen. Ik heb tijdens deze cursus veel geleerd over databases. Na de cursus heb ik nog enkele YouTube tutorials gevolgd over bijvoorbeeld Kali Linux en de tools die daarin verwerkt zaten. Uiteindelijk heb ik niet heel veel met hacken gedaan.
